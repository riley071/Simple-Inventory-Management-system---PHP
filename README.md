# 📦 Simple Inventory Management System (PHP)

A web-based **Inventory Management System** developed using **PHP** and **MySQL** to help businesses efficiently manage inventory, process customer orders, generate invoices, and monitor stock levels in real time.

The system provides a centralized platform for managing products, customers, suppliers, and sales while reducing manual inventory management tasks.

---

# 📖 Overview

The **Simple Inventory Management System** is designed to simplify inventory operations by providing an easy-to-use interface for managing products, customers, suppliers, and orders.

The application enables administrators to:

- Manage inventory in real time
- Process customer orders
- Generate invoices
- Monitor stock availability
- Generate inventory reports
- Manage suppliers and customers
- Send invoices via email
- Accept digital payments

---

# 🚀 Functional Requirements

## 👥 User Registration

The system allows administrators to register and manage:

- Customers
- Suppliers
- System users

---

## 📦 Inventory Management

Administrators can:

- Add new products
- Update product information
- Monitor stock levels
- Remove discontinued products
- Track inventory availability in real time

---

## 🛒 Order Processing

The system enables administrators to:

- Process customer orders
- Generate invoices automatically
- Record completed transactions
- Print invoices

---

## 🚚 Supplier Management

The supplier module allows administrators to:

- Register suppliers
- Update supplier information
- Generate purchase orders
- Manage supplier records

---

## 📊 Reports

Generate inventory reports based on selected dates.

Reports can be:

- Viewed
- Printed
- Used for inventory analysis

---

## 🧾 Invoice Generation

The system automatically generates invoices for customer orders.

Features include:

- Printable invoices
- Order summaries
- Transaction records

---

## 📧 Email Notifications

The application supports sending invoices directly to customers via email after successful order processing.

---

## 💳 Payment Processing

Customers can complete payments by scanning a payment QR code, providing a simple and convenient payment experience.

# 🚀 Installation

### 1. Install XAMPP

Download and install XAMPP from:

https://www.apachefriends.org/

---

### 2. Download the Project

Clone the repository:

```bash
git clone https://github.com/riley071/Simple-Inventory-Management-system---PHP.git
```

Or download the ZIP file and extract it.

---

### 3. Move the Project

Copy the extracted project folder into:

```
C:\xampp\htdocs\
```

---

### 4. Create the Database

Open:

```
http://localhost/phpmyadmin
```

Create a new database named:

```
Whole_db
```

---

### 5. Import the Database

Import the SQL file provided inside the **SQL** folder.

---

### 6. Configure the Database

Update your database connection settings if necessary.

---

### 7. Run the Application

Open your browser and navigate to:

```
http://localhost/Inventory-System/
```
